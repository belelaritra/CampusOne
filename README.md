# 🎓 Campus One (React + Django)

Frontend for **IITB Student Portal** created.

---

### Frontend

* **React (Vite)** – Fast frontend build tool
* **Axios** – API communication
* **JavaScript (ES6+)**
* **CSS / Tailwind (optional upgrade)**



# FRONTEND SETUP (React)

### Install dependencies

```bash
cd frontend
npm install
```

###  Run frontend

```bash
npm run dev
```

### Frontend URL

```
http://localhost:5173/
```
